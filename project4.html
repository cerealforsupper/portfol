Physics Performance Task

In this performance task, my team and I had to build an antenna meant for communication in post-disaster conditions. We chose a dipole design as it was quick and easy to set up, but yet still effective, which was vital for such situations.

For testing, we had to receive another team's transmission and decode their Morse code message using Python. We also transmitted a message to other teams after encrypting it in Morse code. An analysis of the signal was run through the CubicSDR software.
